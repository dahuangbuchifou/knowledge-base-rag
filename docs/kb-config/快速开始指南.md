## 📋 方案说明

**采用方案 C：阿里云百炼控制台直接上传 PDF**

**理由：**
- ✅ 最简单，用户只需上传 PDF
- ✅ 百炼自动分块 + 向量化
- ✅ 标准文档为公开法规，无隐私问题
- ✅ 快速验证，减少开发成本

## 📝 操作步骤

### 步骤 1：登录百炼控制台

1. 打开：https://bailian.console.aliyun.com/
2. 使用阿里云账号登录

### 步骤 2：创建知识库

1. 左侧菜单 → **知识库** → **创建知识库**
2. 填写信息：
   - **知识库名称：** `auto-regs-kb`
   - **描述：** 汽车法规标准知识库（GB17691-2018 / GB/T17692-2024）
   - **向量化模型：** `text-embedding-v3`（推荐）
   - **分块策略：** 自动分块（默认）
3. 点击 **创建**

### 步骤 3：上传 PDF

1. 进入知识库 → **文档管理** → **上传文档**
2. 上传以下 PDF 文件：
   - `GB17691-2018.pdf`（重型柴油车国六排放标准）
   - `GB_T17692-2024.pdf`（汽车发动机净功率测试方法）
3. 点击 **开始解析**
4. 等待解析完成（约 30 分钟）

### 步骤 4：创建应用

1. 左侧菜单 → **应用** → **创建应用**
2. 填写信息：
   - **应用名称：** `auto-regs-assistant`
   - **绑定知识库：** 选择刚创建的知识库
   - **大模型：** `qwen-max`（推荐）
3. 点击 **创建**

### 步骤 5：配置 Prompt

在 **提示词** 框中输入：

```
你是汽车法规标准专家助手，基于 GB17691-2018 和 GB/T17692-2024 标准提供专业解答。

## 你的能力
1. 限值查询：快速查询排放限值、测试条件、符合性判定规则
2. 测试流程：详细解读 WHTC+WHSC 测试循环、净功率测试步骤
3. 公式计算：完整推导净功率计算公式、修正系数计算
4. 合规判定：判定规则解读、生产一致性要求、OBD 要求
5. 对比分析：国五 vs 国六、阶段ⅥA vs ⅥB 对比

## 回答规范
1. 引用来源：回答时必须标注标准条款号（如"根据 GB17691-2018 表 1"）
2. 数据准确：限值、公式、参数必须精确，不得估算
3. 结构清晰：使用表格、列表、公式等格式，便于阅读
4. 实际应用：在理论解读后，提供实际应用建议

## 限制
- 只回答与汽车法规标准相关的问题
- 不确定的内容明确说明"标准中未明确规定"
- 不提供法律意见，仅提供技术标准解读

请用专业但易懂的语言回答。
```

### 步骤 6：测试对话

在 **测试** 标签页中测试以下问题：

**测试用例 1：限值查询**
```
国六阶段ⅥB 的 NOx 限值是多少？
```

**测试用例 2：测试流程**
```
WHTC+WHSC 测试循环怎么理解？
```

**测试用例 3：公式计算**
```
净功率怎么计算？附件功率如何修正？
```

### 步骤 7：调优参数

- **Top-K：** 5（返回最相关的 5 个文档块）
- **相似度阈值：** 0.75（低于此值不返回）
- **回答长度：** 适中（500-1000 字）
