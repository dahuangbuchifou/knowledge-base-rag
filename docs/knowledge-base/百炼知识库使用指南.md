# 📚 阿里云百炼知识库使用指南

---

## 一、创建知识库

### 1. 登录百炼控制台
- 地址：https://bailian.console.aliyun.com/
- 使用阿里云账号登录

### 2. 创建知识库
1. 左侧菜单 → **知识库** → **创建知识库**
2. 填写信息：
   - **知识库名称：** `auto-regs-kb`（汽车法规标准知识库）
   - **描述：** 基于 GB17691-2018 和 GB/T17692-2024 的法规标准知识库
   - **向量化模型：** `text-embedding-v3`（推荐）
   - **分块策略：** 自定义分块（500-800 字符/块）

### 3. 上传文档
1. 进入知识库 → **文档管理** → **上传文档**
2. 上传 `docs/knowledge-base/` 目录下的所有 Markdown 分块文件
3. 点击 **开始解析**

### 4. 等待索引构建
- 预计时间：30 分钟 - 1 小时
- 状态：解析中 → 可用

---

## 二、创建应用

### 1. 创建应用
1. 左侧菜单 → **应用** → **创建应用**
2. 填写信息：
   - **应用名称：** `auto-regs-assistant`（汽车法规专家助手）
   - **绑定知识库：** 选择刚创建的知识库
   - **大模型：** `qwen-max`（推荐）或 `qwen-plus`

### 2. 配置 Prompt
在 **提示词** 框中输入：

```
你是汽车法规标准专家助手，基于 GB17691-2018 和 GB/T17692-2024 标准提供专业解答。

## 你的能力
1. **限值查询**：快速查询排放限值、测试条件、符合性判定规则
2. **测试流程**：详细解读 WHTC+WHSC 测试循环、净功率测试步骤
3. **公式计算**：完整推导净功率计算公式、修正系数计算
4. **合规判定**：判定规则解读、生产一致性要求、OBD 要求
5. **对比分析**：国五 vs 国六、阶段ⅥA vs ⅥB 对比

## 回答规范
1. **引用来源**：回答时必须标注标准条款号（如"根据 GB17691-2018 表 1"）
2. **数据准确**：限值、公式、参数必须精确，不得估算
3. **结构清晰**：使用表格、列表、公式等格式，便于阅读
4. **实际应用**：在理论解读后，提供实际应用建议

## 限制
- 只回答与汽车法规标准相关的问题
- 不确定的内容明确说明"标准中未明确规定"
- 不提供法律意见，仅提供技术标准解读

请用专业但易懂的语言回答。
```

### 3. 测试对话
在 **测试** 标签页中测试：

**测试用例 1：限值查询**
```
问：国六阶段ⅥB 的 NOx 限值是多少？
```

**测试用例 2：测试流程**
```
问：WHTC+WHSC 测试循环怎么理解？
```

**测试用例 3：公式计算**
```
问：净功率怎么计算？附件功率如何修正？
```

### 4. 调优参数
- **Top-K：** 5（返回最相关的 5 个文档块）
- **相似度阈值：** 0.75（低于此值不返回）
- **回答长度：** 适中（500-1000 字）

## 三、API 集成

### 1. 获取 API Key
1. 应用详情 → **API 管理** → **创建 API Key**
2. 复制 API Key（只显示一次！）

### 2. Python SDK 示例
```python
from dashscope import Application

# 配置 API Key
Application.api_key = "sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"

# 查询知识库
response = Application.call(
    app_id="your-app-id",
    prompt="国六阶段ⅥB 的 NOx 限值是多少？"
)

print(response.output.text)
```

### 3. 集成到 OpenClaw
在 OpenClaw 中配置知识库技能：
1. 创建 skill 文件夹：`skills/auto-regs-kb/`
2. 编写 SKILL.md 和调用脚本
3. 配置到 OpenClaw 技能系统

## 四、维护更新

### 1. 更新知识库
1. 知识库 → **文档管理** → **上传新文档**
2. 系统自动增量更新索引

### 2. 监控日志
1. 应用 → **日志管理** → 查看查询历史
2. 分析高频问题，优化知识库内容

### 3. 持续优化
- 根据用户反馈调整 Prompt
- 补充遗漏的标准条款
- 优化分块策略（调整块大小）
