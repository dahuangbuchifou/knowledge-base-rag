# 📚 知识库分类体系

_创建时间：2026-04-28 | 版本：v1.0 | 负责人：大黄_

---

## 一、设计原则

1. **简洁实用**：一级分类 3-5 个，避免过度细分
2. **互斥性**：每个文档只属于一个分类
3. **可扩展**：预留扩展空间，后续可平滑增加
4. **面向检索**：分类服务于 RAG 检索，便于语义匹配

---

## 四、标签体系

### 标签分类

| 标签类别 | 标签 | 说明 |
|----------|------|------|
| **标准编号** | `GB17691`、`GB17692` | 标准号简称 |
| **阶段** | `国六ⅥA`、`国六ⅥB`、`国五` | 排放标准阶段 |
| **污染物** | `NOx`、`PM`、`CO`、`HC`、`NH3` | 污染物种类 |
| **测试类型** | `WHTC`、`WHSC`、`NET_POWER` | 测试循环/方法 |
| **车辆类型** | `重型柴油车`、`重型汽油车`、`轻型车` | 适用车型 |
| **内容类型** | `限值表`、`公式`、`流程图`、`对比表` | 文档内容形式 |
| **知识领域** | `排放`、`功率`、`OBD`、`生产一致性` | 技术领域 |

### 标签使用规范

1. 每个文档标注 2-4 个标签
2. 优先使用标准编号标签（必选）
3. 污染物标签用化学式（NOx 而非 氮氧化物）
4. 阶段标签统一用"国六ⅥA"格式（不用"国六B"等别名）
5. 避免同义词：用 `NOx` 不用 `氮氧化物`，用 `WHTC` 不用 `瞬态循环`

### 标签映射（同义词统一）

| 统一标签 | 同义词（映射到此标签） |
|----------|----------------------|
| `NOx` | 氮氧化物、NO_X |
| `PM` | 颗粒物、particulate |
| `国六ⅥA` | 国六A、阶段ⅥA、VI-A |
| `国六ⅥB` | 国六B、阶段ⅥB、VI-B |
| `WHTC` | 瞬态循环、World Harmonized Transient Cycle |
| `WHSC` | 稳态循环、World Harmonized Stationary Cycle |

## 六、维护规则

1. **新增文档**：先确定分类，再按模板填写
2. **标签审核**：新增标签需确认无同义词冲突
3. **定期审查**：每季度审查一次分类合理性
4. **版本管理**：分类体系变更需更新版本号
