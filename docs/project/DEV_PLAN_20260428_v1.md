## 🔴 P0 — 对话能力增强（核心体验）

### 1.1 多轮对话上下文管理
- **功能：** 支持追问、上下文记忆、会话历史
- **技术方案：** 
  - 百炼 API `session_id` 会话保持
  - Redis 存储短期记忆（可选）
  - 前端会话状态管理
- **预计耗时：** 2-3 小时
- **依赖：** 无

### 1.2 意图识别与路由
- **功能：** 识别用户意图，路由到不同处理模块
- **技术方案：**
  - 简单规则匹配（关键词）
  - 百炼 API 自带意图识别
  - 可扩展：小模型分类器
- **预计耗时：** 1-2 小时
- **依赖：** 无

### 1.3 RAG 检索优化
- **功能：** 提升检索准确率
- **技术方案：**
  - 查询改写（Query Rewriting）
  - 多路召回（关键词 + 向量）
  - 重排序（Rerank）
- **预计耗时：** 2-3 小时
- **依赖：** 需了解百炼 API 能力

### 1.4 回答引用溯源
- **功能：** 展示答案来源（文档名 + 页码）
- **技术方案：**
  - 百炼 API 开启 `has_thoughts: true`
  - 前端展示引用卡片
  - 点击展开原文片段
- **预计耗时：** 1-2 小时
- **依赖：** 百炼 API 支持

### 1.5 多模态支持（可选）
- **功能：** 图文/语音/文件解析
- **技术方案：**
  - 图片上传 + OCR
  - 语音输入（浏览器 API）
  - PDF/Word 文件上传解析
- **预计耗时：** 4-6 小时
- **依赖：** 需额外 API

### 1.6 兜底机制与质量评测
- **功能：** 低质量回答处理
- **技术方案：**
  - 置信度阈值判断
  - 低置信度时返回"不确定"
  - 用户反馈收集（点赞/点踩）
- **预计耗时：** 1-2 小时
- **依赖：** 无
