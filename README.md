## 💡 解读示例

### 示例 1：限值查询
```
问：国六阶段ⅥB 的 NOx 限值是多少？
答：根据 GB17691-2018 表 1，重型柴油车阶段ⅥB 的 NOx 限值为：
    - WHTC 循环：350 mg/kWh
    - WHSC 循环：350 mg/kWh
    （对比阶段ⅥA：460 mg/kWh，降低了 23.9%）
```

### 示例 2：测试流程
```
问：WHTC+WHSC 测试循环怎么理解？
答：根据 GB17691-2018 附录 C：
    1. WHTC（World Harmonized Transient Cycle）：
       - 模拟实际道路瞬态工况
       - 持续时间：1800 秒（30 分钟）
       - 包含冷启动和热启动阶段
    2. WHSC（World Harmonized Stationary Cycle）：
       - 稳态工况测试
       - 13 个稳态工况点
       - 用于验证稳态排放
```
